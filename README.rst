=====
About
=====

A multi-engine part-of-speech tagging system.

Brings together the following engines:

* Adobe cube NLP
* Jieba
* PyKomoran
* PyThaiNLP
* pyvi
* Spacy
* Stanford NLP

Not ready for general use - built for use at http://langlynx.com
